My Creative Process

These are some initial steps that cover the planning and development phase of the project. The other phases of project code delivery, review and maintenance were not described here

1

Discuss The Project

At this stage, a discussion takes place with the client about the project to be developed.

2

Brainstorming & Needs

I hold a meeting to align ideas and create project concepts and needs. At this stage I also check whether the client has good photographs and the company logo.

3

UI/UX Planning

All visual planning (Links, buttons, colors, typography, grid sections, carousel, cards and others) is carried out on some wireframing platform.

4

Development & Interaction

With all the requirements and resources already defined, the code development phase begins and some meetings are held to demonstrate the progress of the project.

After the development phase, the verification of all code and requirements begins to begin the phase of delivering the project to the client. The entire site is optimized (images, code and features).
